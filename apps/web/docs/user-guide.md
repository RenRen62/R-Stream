## インストール手順

### システム要件

1. 推奨環境

   - OS: Windows 10/11, macOS 10.15以上, Ubuntu 20.04以上
   - ブラウザ: Chrome 100以上, Firefox 100以上, Safari 15以上
   - 解像度: 1920x1080以上
   - メモリ: 8GB以上
   - ストレージ: 1GB以上の空き容量

2. 必要なソフトウェア
   - Node.js v18以上
   - npm v8以上
   - Git

### インストール手順

1. リポジトリのクローン

   ```bash
   git clone https://github.com/RenRen62/R-Stream.git
   cd R-Stream
   ```

2. 依存関係のインストール

   ```bash
   npm install
   ```

3. 環境変数の設定

   ```bash
   cp .env.example .env
   # .envファイルを編集して必要な設定を行う
   ```

4. アプリケーションの起動
   ```bash
   npm run dev
   ```

## 基本的な使い方

### セッションの開始

1. スピーカーとして開始

   - 「新規セッション」をクリック
   - セッションタイトルを入力
   - 「開始」をクリック

2. リスナーとして参加
   - セッションIDを入力
   - 表示名を設定
   - 「参加」をクリック

### 理解度の表示

1. キーボード操作

   - 左矢印キー: 理解度低下
   - 右矢印キー: 理解度上昇
   - キーを離す: 通常状態に戻る

2. 画面操作
   - 理解度ボタンをクリック
   - スライダーで調整
   - 「リセット」で通常状態に戻る

### 質問の送信

1. キーボード操作

   - Qキー: 質問モード開始
   - Enter: 質問送信
   - Esc: 質問モード終了

2. 画面操作
   - 「質問をしたい」ボタンをクリック
   - 質問文を入力
   - 「送信」をクリック

## よくある質問

### 一般

1. Q: セッションIDはどこで確認できますか？
   A: スピーカー画面の右上に表示されています。

2. Q: 匿名で参加できますか？
   A: はい、リスナー設定で匿名モードを選択できます。

3. Q: 録画データはどこに保存されますか？
   A: ローカルストレージとクラウドストレージの両方に保存されます。

### 技術

1. Q: 対応ブラウザは何ですか？
   A: Chrome、Firefox、Safariの最新版に対応しています。

2. Q: オフラインでも使用できますか？
   A: 基本的な機能はオフラインでも使用可能ですが、一部機能は制限されます。

3. Q: データのバックアップは自動ですか？
   A: はい、定期的に自動バックアップが実行されます。

## キーボードショートカット

### 共通

- F1: ヘルプ表示
- F11: 全画面表示
- Esc: モード終了

### スピーカー

- R: 録画開始/停止
- S: セッション設定
- A: アーカイブ表示

### リスナー

- ←: 理解度低下
- →: 理解度上昇
- Q: 質問モード
- Enter: 質問送信
