# @r-stream/web

Webアプリケーションのサブプロジェクトです。

## ディレクトリ構造
```
src/
  ├── assets/          # 画像やフォントなどの静的ファイル
  ├── components/      # 共通コンポーネント
  │   ├── common/     # 汎用的なコンポーネント
  │   └── layouts/    # レイアウト関連のコンポーネント
  ├── features/        # 機能ごとのコンポーネントと関連ファイル
  ├── hooks/          # カスタムフック
  ├── types/          # 型定義ファイル
  ├── utils/          # ユーティリティ関数
  ├── constants/      # 定数定義
  └── styles/         # グローバルスタイルやテーマ設定
```
